**✨ IMGBB Uploader Bot Features**

| Feature | Description | Status |
|---------|-------------|--------|
| 🖼️ High Resolution | Auto-selects largest photo quality from Telegram | ✅ Active |
| ⚡ Instant Upload | Average 1.8s upload time to ImgBB | ✅ Active |
| 🌐 Permanent Links | Images hosted forever (no expiration) | ✅ Active |
| 📏 32MB Support | Full ImgBB file size limit supported | ✅ Active |
| 📱 Mobile Optimized | Perfect UX on all Telegram clients | ✅ Active |

***⚙️ Configuration Variables***

Add these environment variables to configure your bot:
| Variable           | Description                                                |
|--------------------|------------------------------------------------------------|
| `BOT_TOKEN`        | Bot Token from [@BotFather](https://t.me/BotFather)         |
| `IMGBB_API_KEY`         | Get it from api.imgbb.com</a>                           |
